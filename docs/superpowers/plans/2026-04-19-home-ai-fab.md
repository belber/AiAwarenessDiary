# Home AI FAB Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the large home AI entry card with a compact floating circular action button attached to the input bar, and move AI feedback to snackbar.

**Architecture:** Keep the existing AI action routing in the home screen model layer, move transient feedback into snackbar-oriented helpers and `MainViewModel` state, then replace the bottom card with a floating button rendered alongside the input bar. This limits behavior changes to the home screen while preserving generation flow.

**Tech Stack:** Kotlin, Jetpack Compose, Material 3, JUnit
